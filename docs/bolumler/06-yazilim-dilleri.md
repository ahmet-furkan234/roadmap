# Yazılım Dilleri

Programlama dilleri bir araçtır. Her araç bir iş için tasarlanmıştır.

Tıpkı çekiç ve tornavidanın farklı işler için var olması gibi, her dil belirli bir alanda güçlüdür. Hangisini öğreneceğini bilmeden önce hangisinin ne işe yaradığını bilmek gerekir.

---

## Genel Bakış

```
Web Frontend    →  JavaScript / TypeScript
Web Backend     →  Python, JavaScript (Node.js), Java, PHP, Go, Ruby
Mobil (iOS)     →  Swift
Mobil (Android) →  Kotlin
Oyun            →  C#, C++
Veri Bilimi     →  Python
Sistem / Gömülü →  C, C++, Rust
```

---

## Diller ve Kullanım Alanları

### 🐍 Python
**Kullanım:** Veri bilimi, yapay zeka, web backend, otomasyon, scripting
**Zorluk:** ⭐ Başlangıç için en kolay dillerden biri
**Neden öğren:** Okunması kolay, sözdizimi İngilizce'ye yakın, her alanda kullanılıyor. Yazılıma ilk adım için çok tercih edilir.

```python
isim = "Ahmet"
print("Merhaba, " + isim)
```

---

### 🌐 JavaScript
**Kullanım:** Web frontend (tarayıcı), web backend (Node.js), mobil (React Native)
**Zorluk:** ⭐⭐ Orta — başlangıç kolay ama derinleştikçe karmaşıklaşır
**Neden öğren:** Tek dil, her yerde çalışır. Tarayıcıda çalışan tek dildir — web yapmak istiyorsan kaçış yok.

```js
const isim = "Ahmet"
console.log("Merhaba, " + isim)
```

---

### 🔷 TypeScript
**Kullanım:** JavaScript'in her kullanıldığı yer
**Zorluk:** ⭐⭐ JavaScript'i biliyorsan kolayca geçilir
**Neden öğren:** JavaScript'in üzerine tip sistemi ekler. Büyük projelerde hataları erkenden yakalar. Endüstride giderek standart haline geliyor.

---

### ☕ Java
**Kullanım:** Kurumsal uygulamalar, Android, büyük ölçekli sistemler
**Zorluk:** ⭐⭐⭐ Sözdizimi daha katı ve ayrıntılı
**Neden öğren:** Onlarca yıldır büyük şirketlerin omurgası. Banka yazılımları, sigorta sistemleri, kamu kurumları Java kullanır. İş imkânı çok.

```java
String isim = "Ahmet";
System.out.println("Merhaba, " + isim);
```

---

### 🔵 C\#
**Kullanım:** Windows uygulamaları, oyun geliştirme (Unity), kurumsal backend
**Zorluk:** ⭐⭐⭐ Java'ya benzer yapı
**Neden öğren:** Unity oyun motoru C# kullanır — oyun geliştirmeye ilgi duyuyorsan bu dil. Microsoft ekosistemiyle de derin entegrasyonu var.

---

### 🟠 Swift
**Kullanım:** iOS ve macOS uygulama geliştirme
**Zorluk:** ⭐⭐ Modern ve okunabilir sözdizimi
**Neden öğren:** iPhone veya Mac uygulaması yapmak istiyorsan tek gerçekçi seçenek bu.

---

### 🟢 Kotlin
**Kullanım:** Android uygulama geliştirme, backend (Spring)
**Zorluk:** ⭐⭐ Java'ya göre daha modern ve az kod
**Neden öğren:** Google'ın Android için önerdiği resmi dil. Java'nın yerini almaya devam ediyor.

---

### 🔵 Go (Golang)
**Kullanım:** Backend, mikroservisler, DevOps araçları
**Zorluk:** ⭐⭐ Basit sözdizimi ama farklı düşünme biçimi gerektirir
**Neden öğren:** Yüksek performanslı, hızlı derlenen modern bir dil. Bulut altyapısı ve büyük ölçekli sistemlerde artıyor.

---

### ⚙️ C / C++
**Kullanım:** İşletim sistemleri, oyun motorları, gömülü sistemler, donanım yazılımı
**Zorluk:** ⭐⭐⭐⭐⭐ En zorlu dillerden biri
**Neden öğren:** Diğer dillerin altında yatan temeli anlamak için. Performansın en önemli olduğu alanlarda hâlâ kral. Başlangıç için önerilmez ama ileri seviyede çok değerli.

---

### 🦀 Rust
**Kullanım:** Sistem programlama, WebAssembly, yüksek performanslı uygulamalar
**Zorluk:** ⭐⭐⭐⭐⭐ Öğrenmesi en zor dillerden biri
**Neden öğren:** C/C++'ın güvenlik açıklarını kapatan modern alternatif. Geleceğin sistem dili olarak öne çıkıyor.

---

## Hangisini Seçmeliyim?

| Hedefin | Önerilen Dil |
|---|---|
| Web sitesi yapmak | JavaScript |
| Mobil uygulama (Android) | Kotlin |
| Mobil uygulama (iOS) | Swift |
| Veri bilimi / yapay zeka | Python |
| Genel amaç, ilk dil | Python veya JavaScript |
| Oyun geliştirme | C# (Unity) |
| Kurumsal yazılım | Java veya C# |
| Sistem / donanım | C veya Rust |

---

## Önemli Not

Bu listeyi gördüğünde "hangisini seçeyim?" diye paniklemene gerek yok.

Şunu bil: **ilk dili öğrendiğinde ikincisi çok daha kolay gelir.** Diller arasında geçiş yapmak, sıfırdan başlamak gibi değildir. Mantık aynı, sadece sözdizimi değişiyor.

Başlangıç için Python veya JavaScript seç. Birini iyi öğren. Gerisini zaman gösterir.
